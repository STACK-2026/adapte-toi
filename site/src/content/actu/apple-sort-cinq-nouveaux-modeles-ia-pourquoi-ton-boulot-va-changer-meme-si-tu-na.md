## Le fait
Apple a dévoilé sa troisième génération de modèles d'IA, les *Apple Foundation Models 3* (AFM 3), lors de la WWDC 2026. La marque reconnaît que ses premiers modèles (2025) étaient "en retrait", selon [Next.ink](https://next.ink/241767/apple-detaille-ses-cinq-nouveaux-modeles-dia-et-admet-a-demi-mot-ses-rates/).

Apple Intelligence repose sur cinq modèles :
- **Deux modèles locaux** : AFM 3 Core (3 milliards de paramètres) et AFM 3 Core Advanced (20 milliards, mais seulement 1 à 4 milliards activés simultanément).
- **Trois modèles cloud** : AFM 3 Cloud (vitesse), ADM 3 Cloud (génération d'images), et AFM 3 Cloud Pro (requêtes complexes, hébergé sur des GPU NVIDIA dans Google Cloud).

AFM 3 Core Advanced nécessite **12 Go de RAM** : seuls l'iPhone 17 Pro, l'iPhone Air, les Mac M3 et les iPad M4 sont compatibles. Les autres appareils utilisent le modèle de base.

Les benchmarks internes montrent une amélioration : AFM 3 Core est préféré à son prédécesseur dans **45,6 % des cas** pour le texte local, et **61 %** pour la compréhension d'images. En cloud, AFM 3 Cloud obtient **64,7 % de préférences** contre 8,7 % pour l'ancienne génération.

## Ce qu'on en dit vraiment
Apple admet ses lacunes initiales. La première génération d'Apple Intelligence (2025) était insuffisante, poussant la marque à accélérer le développement d'AFM 3. La **mémoire dynamique** (activation partielle des paramètres) permet de contourner les limites matérielles, une approche potentiellement utile pour d'autres outils pro.

L'IA locale n'est plus un gadget. AFM 3 Core Advanced permet d'exécuter un modèle de 20 milliards de paramètres sur un iPhone, sans cloud. Un atout pour les métiers nécessitant la confidentialité (avocats, médecins), mais réservé aux appareils récents.

Les modèles cloud dépendent de Google et NVIDIA. AFM 3 Cloud Pro tourne sur des GPU NVIDIA hébergés dans Google Cloud, avec une extension *Private Cloud Compute* pour sécuriser les données. Les outils IA d'Apple (génération d'images, requêtes complexes) rivalisent avec la concurrence, mais avec une latence supplémentaire.

Les benchmarks d'Apple sont biaisés. En comparant AFM 3 uniquement à sa génération précédente, la marque évite les comparaisons avec GPT-4o ou Claude 3.5. Pourtant, les chiffres montrent que le nouveau modèle est moins performant dans **20 à 30 % des cas**. Une amélioration, mais pas une révolution.

## Les chiffres qui comptent
- **3 milliards** : paramètres du modèle local AFM 3 Core (20 milliards pour AFM 3 Core Advanced, mais seulement 1 à 4 milliards activés). ([Next.ink](https://next.ink/241767/apple-detaille-ses-cinq-nouveaux-modeles-dia-et-admet-a-demi-mot-ses-rates/))
- **12 Go de RAM** : requis pour AFM 3 Core Advanced. Les iPhone 17 standard (8 Go) sont exclus. ([Next.ink](https://next.ink/241767/apple-detaille-ses-cinq-nouveaux-modeles-dia-et-admet-a-demi-mot-ses-rates/))
- **45,6 %** : préférence pour AFM 3 Core en texte local (contre 23,3 % pour l'ancien modèle). ([Next.ink](https://next.ink/241767/apple-detaille-ses-cinq-nouveaux-modeles-dia-et-admet-a-demi-mot-ses-rates/))
- **64,7 %** : préférence pour AFM 3 Cloud (contre 8,7 % pour la génération précédente). ([Next.ink](https://next.ink/241767/apple-detaille-ses-cinq-nouveaux-modeles-dia-et-admet-a-demi-mot-ses-rates/))
- **61 %** : préférence pour AFM 3 Core en compréhension d'images. ([Next.ink](https://next.ink/241767/apple-detaille-ses-cinq-nouveaux-modeles-dia-et-admet-a-demi-mot-ses-rates/))

## Pour toi concrètement

Apple utilise la **mémoire dynamique** pour optimiser les performances. Une technique à explorer pour tes outils IA, surtout sur mobile.
- **À 30 jours** : Étudie les [recherches d'Apple](https://machinelearning.apple.com/) sur la sélection dynamique de paramètres.
- **À 60 jours** : Teste AFM 3 Core Advanced sur un iPhone 17 Pro ou Mac M3. Compare avec [Claude 3.5 Haiku](/outils/claude) ou [GPT-4o mini](/outils/chatgpt) pour des tâches locales.
- **À 90 jours** : Évalue l'IA locale d'Apple pour les métiers réglementés. Consulte notre [guide sur les outils IA pour développeurs](/guides/automatiser-travail-ia).

### Graphiste, community manager ou créateur de contenu

- **À 90 jours** : Intègre les voix de synthèse de Siri dans tes vidéos ou podcasts. Consulte notre [fiche métier community manager IA](/metiers/community-manager-ia).
