GitHub Copilot est un assistant de programmation IA créé par GitHub (Microsoft). Il s'intègre à ton éditeur de code (VS Code, JetBrains, Neovim) et te suggère du code en temps réel pendant que tu tapes. C'est comme avoir un collègue développeur qui lit par-dessus ton épaule et complète tes phrases.

## En une phrase

Tu commences à écrire une fonction, Copilot propose le reste. Tu écris un commentaire décrivant ce que tu veux, Copilot génère le code. Précision : 85% au premier essai (Microsoft 2025).

## Pour qui ?

- **Développeurs juniors** : apprendre les bonnes pratiques en voyant le code suggéré
- **Développeurs seniors** : accélérer le code boilerplate, se concentrer sur l'architecture
- **Data analysts** : écrire du SQL et du Python sans mémoriser la syntaxe
- **Étudiants** : gratuit avec le GitHub Student Pack

## Ce que ça fait

- **Auto-complétion contextuelle** : Copilot comprend le fichier en cours et suggère des blocs entiers
- **Code à partir de commentaires** : `// fonction qui trie un tableau par date décroissante` -> code généré
- **Tests unitaires** : sélectionne une fonction, demande les tests, Copilot les écrit
- **Explication de code** : sélectionne du code incompréhensible, Copilot l'explique en français
- **Refactoring** : propose des versions plus propres du code existant

## Démarrage rapide

1. Va sur **github.com/features/copilot** et active l'essai gratuit
2. Installe l'extension "GitHub Copilot" dans VS Code
3. Ouvre un fichier et commence à taper : les suggestions apparaissent en gris
4. Tab pour accepter, Esc pour ignorer

Consulte notre [fiche développeur](/métiers/developpeur-ia) pour comprendre l'impact de l'IA sur le métier de dev.

## Notre verdict : 8/10

GitHub Copilot est le meilleur choix pour les développeurs qui veulent de l'IA sans changer d'éditeur. Moins puissant que [Cursor](/outils/cursor) sur les modifications multi-fichiers, mais plus léger et moins cher. Le plan gratuit pour les étudiants est un cadeau.
